Propósito

Este tema tiene como proposito dar aprendizge y dar a conocer a fondo todo lo que tiene con flex Box y Css Grid, y dar a conocer todo lo que conietiene este, ya que estas herramientas son muy importantes

Introducción

Este trabajo tiene como objetivo ver a fondo los temas de flexbox y css grid ya que son muy importanters a la hora de realizar paginas web. flexbox es un método de diseño de una página unidimensional para poder compaginar elementos en filas o columnas. y Css Grid es importante ya que es un modelo de maquetado que es basado basado en un sistema de rejillas incorporado en CSS3 para poder gestionar de manera mas sencilla y customizable la disposición de nuestros elementos para nuestra página web.
